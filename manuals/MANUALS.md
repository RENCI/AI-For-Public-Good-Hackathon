# Manuals

Reference PDFs for cancer registry coding. These are the authoritative sources for NAACCR data item definitions, staging rules, site/histology validation, and facility reporting standards. Total size: ~23 MB across 6 documents.

These manuals are indexed in an OpenAI vector store and searchable via goodbot's `search_manuals` tool.

## Files

### `fords-2016.pdf` (4.6 MB, 515 pages)

**2016 FORDS: Facility Oncology Registry Data Standards** — American Cancer Society

Defines data collection standards for hospital-based cancer registries. Covers required and recommended data items, coding instructions for each NAACCR field, and rules for abstracting cancer cases from medical records at the facility level.

### `seer_summary_staging_manual.pdf` (3.5 MB, 299 pages)

**SEER Summary Staging Manual - 2000** — NCI SEER

Provides summary staging definitions (in situ, localized, regional, distant) for all cancer sites. Used to assign SEER Summary Stage codes based on extent of disease at diagnosis.

### `site_histology_validation_list.pdf` (680 KB, 388 pages)

**ICD-O-3 SEER Site/Histology Validation List** — NCI SEER

Comprehensive cross-reference of valid ICD-O-3 topography (site) and morphology (histology) code combinations. Used to validate that an assigned histology code is valid for a given primary site.

### `SolidTumorRules_Combined.pdf` (8.0 MB, 505 pages)

**Solid Tumor Rules 2025 Update** — NCI/SEER

Rules for determining how many primary tumors to abstract and how to code histology for solid tumors. Covers counting rules (single vs. multiple primaries), histology coding rules, and site-specific instructions for each organ system.

### `SPCSM_2024_MainDoc.pdf` (1.9 MB, 279 pages)

**SEER Program Coding and Staging Manual 2024** — NCI SEER

The primary coding manual for the SEER program. Contains field-by-field instructions for coding NAACCR data items including date of diagnosis, primary site, laterality, histology, behavior, grade, and staging variables.

### `store-manual-2024.pdf` (4.5 MB, 461 pages)

**STORE Manual 2024** — by Lianne Babaran

Standards for Oncology Registry Entry. Provides detailed abstracting instructions, data item definitions, and coding guidelines for cancer registrars. Covers the full lifecycle of case abstraction from identification through coding and quality control.
